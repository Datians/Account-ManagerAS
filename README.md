# Gestor de cuentas Andriux Streaming

Proyecto Flask con tema oscuro para gestionar cuentas de plataformas de streaming (Netflix, Disney+, etc.).
Incluye autenticación, CRUD de proveedores, clientes y cuentas, y carga masiva por Excel.

## Instalación rápida
```bash
python -m venv venv
source venv/bin/activate  # (Windows: venv\Scripts\activate)
pip install -r requirements.txt
cp .env.example .env
python run.py
```
